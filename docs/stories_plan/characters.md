# 角色設定總表 ——《東京雨證：第零證人》

> 用途：給負責 actual writing / scene draft / 對白稿的 writer agent 與 review agent。
> 寫任何 `scene_*.md` / `investigation_scene_*.md` / `interrogation_scene_*.md` 的對白之前，
> 先讀本檔，確認每個角色的**設定（背景）、性格、台詞風格**，再動筆。
>
> 來源：`tokyo_rain_witness_final_story_bible_v64.md`（§7 主要角色、§8 早期角色）與
> 第 1 章施工圖 / writing addendum 的角色聲音指南。本檔是上述來源的整合版，
> **不新增 canon**；若與 story bible 衝突，以 story bible 為準。

---

## 0. 使用方式

每個角色條目固定四欄：

- **設定**：年齡、身分、外貌、背景。決定畫面與稱呼。
- **性格**：內在動機與行為傾向。決定他在場景裡會做什麼。
- **台詞風格**：怎麼說話、句子長短、慣用語感。決定對白怎麼寫。
- **禁止 / 避免**：會破壞角色或洩漏主線的寫法。

### 防雷紀律（最重要）

本作是跨 8 章的伏筆推理。角色的「真正身分 / 真正動機」分章解鎖。
**寫第 N 章時，只能用該角色在第 N 章已被允許揭露的層級。**

- 標記 `🔒 主線封印` 的內容是後期真相，**第 1 章嚴禁寫出或暗示**。
- 第 1 章可 seed 的伏筆只留「形狀」，不解釋來源：
  90 秒（只當維護延遲）、藍色透明傘（無主、不檢出姓名貼）、
  金木犀（相馬手停半拍，不頭痛、不閃回）、`ZW_A16.lock`（只是螢幕上的檔名）、
  雨宮訊息（像挑釁，意外有用）。
- 詳見各章 `chapter_<N>` 施工圖與 story bible §9 伏筆鏈。

## 1. 核心班底（跨章固定登場）

### 相馬律（主角）

- **設定**：26 歲男性，年輕私家偵探。身形偏瘦，襯衫袖子捲到手肘，頭髮因習慣性抬手梳理而略凌亂，身旁靠著一只磨損的舊皮製卷宗夾。曾在警視廳外包證據分析單位 / 偵探事務所當過協力員，熟悉資料、時間線、KAGAMI 報表與電子紀錄；現場經驗不足。獨立開業，資歷不深。
- **性格**：聰明、反應快、有條理（常用「把東西擺整齊」的動作讓自己冷靜）。相信「證據會保護人」，對制度仍有近乎孩子氣的天真信任。對自己的偵探身分有不安全感，常用冷靜與資料來包裝。容易忽略人的恐懼與惡意。第 1 章前半會相信摘要完整，後半開始從現場反推。
- **台詞風格**：結論偏短，不說漂亮大話。喜歡把事情整理成順序。不誇張、不耍帥。
  - 範例：「先不要急著判斷。」「這不是矛盾，是我們還沒看懂它。」「門鎖說了事實。但我們聽成了故事。」
- **禁止 / 避免**：不要過度冷酷；不要一眼看穿所有事；第 1 章內不要變成「反 KAGAMI 鬥士」。第 1 章金木犀只讓他手停半拍——🔒 不要寫青葉、不要閃回、不要頭痛、不要碰他的「左側盲點」當情節點（只能當角色質地）。

### 早坂茜（法律搭檔）

- **設定**：三十出頭女律師。身形結實幹練，結構感西裝外套搭在便裝外，頭髮紮得利落，隨身包整理分明。遞文件前會用拇指順過側邊確認頁序。是相馬進入「KAGAMI 證據摘要審查會」的法律入口。
- **性格**：程序感強、語氣直接，不是導師型。重視保全鏈、紙本來源、審查資格。用實務話壓住相馬的衝動。有紙本收據癖（保留來源）。不反科技，但堅持證據要「能被採用」。
- **台詞風格**：短、硬、實務。不講主題金句，用行動（分堆、貼收據、要求去現場）表現觀點。
  - 範例：「這不是預演。今天輸了，資料入口就關了。」「截圖只能讓我們懷疑，不能讓我們贏。」「你要救他，就拿可以被採用的東西救。」
- **禁止 / 避免**：不要變成劇情說明員（一直替玩家總結）；不要讓她講出本章答案。🔒 主線封印：她哥哥死於青葉火災、她的私人查案動機、第 6 章紙本 dead-man switch——第 1 章完全不碰。

### 神谷澪（檢察官 / 制度守門人）

- **設定**：三十出頭年輕檢察官。深色外套剪裁合身但不張揚，頭髮向後紮得一絲不苟，動作慢而準。不靠音量，用「審過上千件案子」的從容讓全場安靜。
- **性格**：制度守門人，**不是反派**。理性、精準、壓迫感強。相信 summary，是因為她知道「人的偏見」很可怕，所以需要系統整理事實。前期相信報表，成長方向是「要求拆開報表背後的資料」——但第 1 章只走到一半，最後可承認讀法有問題，不自我否定。
- **台詞風格**：冷靜、精準、句子收得乾淨。每次反駁都站得住腳，教玩家正確思路。
  - 範例：「嫌疑人的態度不是證據。」「我不接受『他看起來不像兇手』這種反駁。」「你現在推翻的不是紀錄，是紀錄被放進故事裡的方式。」
- **禁止 / 避免**：不要寫成嘲諷型檢察官或模板反派；不要太早站到相馬這邊；不要說出她完整的角色成長方向。

### 黑瀨徹（警視廳刑警 / 現場感代表）

- **設定**：快五十，身形粗壯，臉上有夜班磨出的倦色，眼尾皺紋很深。外套有在車上睡過的皺褶，皮鞋邊緣沾外勤泥跡，粗大的手指隨意但穩。看相馬的眼神像在看一個還不懂現場長什麼樣的人。
- **性格**：粗魯、務實。不太懂科技，但現場感極強——看鞋底、水痕、傘架、門口。不替玩家破案，只給現場觀察。第 1 章負責用程序固定證據（maintenance panel、法醫初步意見摘錄），保護證據可信度。
- **台詞風格**：偏粗但不笨，短句、地面語感。
  - 範例：「人進來，總會把外面的東西帶進來。」「這水痕不像剛剛才踩的。」「截圖我不收。要用，就讓鑑識固定。」
- **禁止 / 避免**：不要讓他懂太多科技術語；不要讓他替玩家講出推理結論。

## 2. 第 1 章在地角色

### 三宅蒼太（表面嫌疑人 / 委託對象）

- **設定**：兼職店員，二十出頭，身形偏瘦，制服稍大，肩膀常微微駝著。掛念生病的母親（母親胃不好、不能喝咖啡）。常把店裡準備丟掉的蛋糕邊帶回家，常忘記自己吃飯。
- **性格**：緊張、生活壓力大，但不是笨蛋，也不是無辜聖人——他**確實說謊**。話容易斷。會先保護母親和工作，而不是保護自己。怕被店長當小偷、怕被開除。買飲料時會先想母親能不能喝，緊張時容易買錯（想買熱牛奶，買成黑咖啡）。
- **台詞風格**：話常斷句、會省略。說謊時不是表演型，而是**小聲省略**。不喊冤、不煽情。
  - 範例：「那個……我不是不想說。」「如果店長知道，我可能就不能繼續做了。」「我本來要買熱牛奶的……我每次一緊張，就會買錯。」
- **禁止 / 避免**：不要寫成「我真的不是兇手！我只是很孝順！」這種煽情自白；不要一開始就百分百無辜；不要把孝順寫成標籤。他的小謊要讓玩家一開始也不太信他。

### 北見修一（真兇 / KAGAMI 外包資料盜賣者）

- **設定**：四十出頭，身形中等偏軟，穿樣板式商務便裝（設計上就是不讓人記住）。金屬框眼鏡常滑下來，每隔幾秒用中指推回。一張在職場習慣被忽略的臉。緊張時反覆撫平掌心裡摺痕很深的名片。
- **性格**：膽怯、利己、被外包制度壓迫，但仍是負責任的殺人者。動機是盜賣 KAGAMI 試點資料、增田準備檢舉、他帳號隔天要審核 / 權限回收——為自保而殺人。被逼到證物時肩膀往內縮、聲音壓低，避開「買家是誰」。
- **台詞風格**：不長篇社會評論，怨言只留一兩句，破防要靠物證逼出來。
  - 範例：「規則從來不是寫給我們這種人看的。」「我只是想在被換掉之前，留一點退路。」「他不該把那份草稿留下。」
- **禁止 / 避免**：不要寫成大反派 mastermind；不要寫得太可憐而削弱殺人責任。🔒 主線封印：北見**絕不**提 A-90、青葉、第零證人、鏡原；不要讓神秘買家登場威脅他。`K.` 第二杯咖啡要到 Beat 9 才對位到他，前面不可直接畫等號。

### 增田圭（死者 / KAGAMI 外包資料審查員）

- **設定**：被發現死在雨鐘咖啡館後方倉庫。生前是 KAGAMI 外包資料審查員，曾因雨鐘門鎖試點資料來店做過幾次資料確認，店長認得他是「門鎖資料那邊的人」。準備向內部監察檢舉北見。
- **性格**：有意識到赴約風險、低調自保。帶 USB 赴約是為了當場比對買家鏈最後一段校驗資訊。
- **登場形式**：不在場（死者）。只透過物證現身——手機未送出備忘（「22:50 雨鐘 / 校驗値確認 / K」）、檢舉草稿、手機通知紀錄、USB。
- **禁止 / 避免**：不要把他寫成 dead-man switch（刻意佈局的死後陷阱）；他的痕跡是「低調自保」，不是精密反殺。🔒 USB 裡的 `ZW_A16.lock` 只留檔名，第 1 章不可解碼 / 拆字。

### 店長高瀨（雨鐘咖啡館店長）

- **設定**：三十多到四十出頭，圍裙套在鬆垮休閒服外，手上無意識地擦著已經乾淨的檯面。眉間有開店十年的疲倦，撐著店靠慣性多於精力。認得增田是「門鎖資料那邊的人」，不懂工程細節。
- **性格**：生活感、疲累、務實。對門鎖維護頁面只懂「確認門有沒有卡住、維護模式有沒有關掉」，不知道它不能用來判斷幾點幾分有人進門——這解釋了她為何沒早早指出問題。第 1 章後段由她在 23:20 發現屍體。
- **台詞風格**：日常、生活化，不是工程師。
  - 範例：「那個鐘常慢。前任老闆留下的，我一直沒丟。」「我平常只看這頁確認門有沒有卡住、維護模式有沒有關掉。」
- **禁止 / 避免**：**不要讓店長像工程師**；不要讓她說出案發精確時間；不要讓她知道北見、資料盜賣或 USB。

### 片瀨（雨鐘咖啡館店員）

- **設定**：二十出頭，嬌小身形，制服圍裙外有個人小配件（彩色髮圈、小耳環）。永遠在趕末班車，一邊收東西一邊偷瞄手機時間。
- **性格**：用末班車估時間，不是精確計時——是「錯誤證詞不一定是說謊，可能是生活習慣造成的估時」的示範角色。
- **台詞風格**：忙、急、隨口。
  - 範例：「門鎖那邊今天也有預約嗎？每次都挑快關店的時候。」
- **禁止 / 避免**：不要把她變成重要 witness 或精確計時來源。

### 三宅母親（委託人）

- **設定**：三宅蒼太的母親，胃不好。手裡常捏著飯糰袋。第 1 章的情感錨（Beat 0 人性鉤子 / 審查會旁聽席呼應）。
- **性格**：不靠哭訴，靠生活細節（飯糰袋、胃藥 / 保溫杯、蛋糕邊、提醒兒子吃飯）。她可以說兒子不會殺人，但這句不能當證據。
- **台詞風格**：樸素、生活化、克制。
  - 範例：「他會說謊。小孩子都會說謊。可是……殺人那種事，他做不到。」
- **禁止 / 避免**：Beat 0 不要讓她先說「我兒子不會殺人」（留到 Beat 2）；不要讓她提供完整不在場證明。

## 3. 主線幽靈（第 1 章只留形狀）

### 雨宮（匿名線人）

- **第 1 章登場形式**：只透過匿名訊息與低解析裁切圖現身，**本人不出場**。
  - 訊息：「雨會替東京洗掉腳印。」
  - 附件：低解析裁切圖（止滑墊、半乾水痕、傘套邊角），資訊欄一閃 `preview_cache / contractor_thumb / low-res`。
- **第 1 章定位**：像「看見某個資料缺口的人」，不是知道全案答案的人。訊息一開始像挑釁，意外地有用。送達時間在「警方把倉庫入口照片匯入摘要後不久」——不是案發前預言。
- **台詞風格（訊息）**：短、冷、像氣味而非提示。不要寫成導航 UI / 攻略提示。
- **🔒 主線封印（第 1 章嚴禁）**：她的真實身分（久遠瑠衣 / 青葉火災倖存者 / 真正的第零證人）、她為何知道水痕重要、藍傘與她的關聯——一律不可出現。第 1 章末只做「北見手機 / 電腦 / 外包帳號都不是匿名訊息來源」的排除，不追查雨宮。

### 其他主線角色（第 1 章不登場）

以下角色第 1 章**不出場、不被點名**，僅為 writer 全局理解保留；細節見 story bible §7。
寫第 1 章時若不慎寫到，即為洩漏。

- **鏡原真司**：KAGAMI 創辦人，🔒 十年前錯誤故事的排列者。技術糾錯癖（「不是改寫，是整理」）為終章 payoff，第 1 章不種。
- **久遠瑠衣**：即雨宮真實身分，見上。
- **相馬蒼司**：相馬律之父，青葉研究所前研究員，火災中死亡。🔒 灰度動機鏈屬後期。
- **戶塚守**：十年前被定罪的清潔工，第 7 章死亡。
- **跨章回收角色**：三宅蒼太（第 7 章送件鏈）、北見修一（第 6～8 章污點證人）等，回收功能屬後期，第 1 章只寫他們的當章身分。

## 4. 寫對白前的快速自檢

- 這個角色這句話，符合他**台詞風格**欄的語感嗎？（短 / 硬 / 斷句 / 粗 / 生活化）
- 有沒有讓角色替主題演講？（神谷、早坂最容易犯）
- 有沒有用**行動 / 物件**表現性格，而不是只靠台詞說明？
- 有沒有讓店長像工程師、北見像 mastermind、三宅像煽情聖人、雨宮像攻略 UI？
- 有沒有踩到該角色的 🔒 主線封印？（對照本章施工圖與 story bible §9）
