# Scene 8.5: 短暫誤判整理點

[場景：吉祥寺商店街屋簷下，深夜，雨仍未停。屋簷邊掛著一塊臨時白板，旁邊放著相馬從事務所帶來的資料夾，雨聲蓋過遠處的人聲。]
- **Background Prompt:** Kichijoji shopping street under an awning at rainy night, temporary whiteboard and case folder set up for evidence sorting, wet street glow, no readable board text.

**早坂茜**：先停一下。卡片翻太快了。

**早坂茜**：在繼續之前，我要把現在手上的東西分一分。

[早坂在白板上劃出三條直線，分成三欄。]

**早坂茜**：第一欄，三宅小謊。第二欄，第三者動線。第三欄，門鎖時序。

[黑瀨從巷口走來，把一份列印紙遞給早坂。]

**黑瀨徹**：補一件程序上的東西。三宅母親那通電話，手機履歷對上了。

**黑瀨徹**：通信紀錄已經核對完，這條不再是待確認，是正式紀錄了。

**早坂茜**：好。那我把它從問號那邊，移到三宅小謊這一欄。

[早坂把那張紙貼進第一欄。]

**早坂茜**：相馬，你來說。現在到底證明了什麼，一條一條講。

**相馬律**：第一，三宅確實說了謊，但理由是那通電話和那個蛋糕盒。

**相馬律**：母親通話紀錄現在被正式固定了，這個謊有了底。

**相馬律**：第二，三宅二十三點零六分站的位置，根本看不到屍體。

**相馬律**：第三，有人比三宅更早進來，走的是承包商那條路。

**相馬律**：第四，現場實際發生的順序，跟摘要上的時間對不上。

[相馬把資料夾合上，看著三欄。]

**早坂茜**：所以你看出來了。

**早坂茜**：我們現在只證明三宅不該被放在那個時間。

**相馬律**：還沒證明誰該被放回去。

**早坂茜**：對。所以接下來不是找更大的謎，是把那個空位填上。

[相馬點頭，目光停在第二欄的空白處。]

**相馬律**：那個位置現在還是空的。比三宅更早進來的人，我們連名字都沒有。

**早坂茜**：名字之後再說。先把能拉到他的線找齊。

[早坂在第二欄底下，寫了三個待辦。]

**早坂茜**：外包工單，先去調。承包商那條路是誰排的，工單上會有。

**相馬律**：增田那份檢舉草稿也得找。他想揭的事，跟這條動線可能是同一件。

**早坂茜**：還有那個 K.。到現在只有一個字母，這次要把它追到底。

[相馬收起資料夾，雨水順著屋簷滴下。]

**相馬律**：工單、草稿、K.。三條線，把空位填上。

**早坂茜**：走吧。趁雨還沒小。
