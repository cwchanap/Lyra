## 0. 使用方式

Writer agent 不要把本文件寫成小說正文。它是每個 beat 的「施工圖」。

每一 beat 需要產出以下內容：

1. **Scene List**：這個 beat 拆成幾個 scene，每個 scene 的地點、角色、目的。
2. **Scene Draft**：主要對話與情緒流。
3. **Interaction Script**：玩家可調查、可出示、可選錯的點。
4. **Evidence Update**：正式證物欄、人物筆記、背景符號各有哪些變化。
5. **Exit Condition**：玩家完成什麼理解後，才能進下一 beat。

本章的寫作目標不是把所有邏輯都用台詞講清楚，而是讓玩家透過場景、操作、視線、物件與審查會選擇，自己走到結論。

**時長 canon：** 本章標準遊玩時間鎖定為 **2.5～3 小時**。以下每 beat 的預估時長是 actual writing 的施工預算，不是可無限延伸的上限。若單 beat scene draft 超時，優先刪減重複台詞與重複調查點，不刪 fair-play 必要因果。

## V1.2 最終補丁鎖定

本版已吸收最後 self-review 的七項防滑補丁：

## 1. 全章前台證據包

第 1 章雖然有多個證物，但前台只能讓玩家記住三個大問題。Writer agent 在每一 beat 都要確認當前內容服務哪個證據包。

| 證據包 | 玩家要理解的命題 | 主要 beat |
|---|---|---|
| 三宅小謊包 | 三宅說謊，但不是因為殺人 | Beat 2、4、5、8.5、10 |
| 第三者動線包 | 有人比三宅更早從承包商動線進入後場 | Beat 3、7、8、8.5、9 |
| 門鎖時序包 | 門鎖沒有說謊，但 summary 把合併時間當成事件時間 | Beat 0、5、8、10 |

寫作上不要讓玩家一次背二十個證物名稱。每次互動只問一個小問題：

- 三宅為什麼說謊？
- 三宅站的位置看得到屍體嗎？
- 有沒有第三者？
- 那個第三者走的是哪條動線？
- 門鎖上的時間到底是什麼時間？
- 誰最符合那個空位？

## 2. 全章口語規則

角色對白只用這三個口語詞：

| 口語詞 | 意義 |
|---|---|
| 摘要 | KAGAMI summary，警方與審查會先看到的整理後故事 |
| 本機順序 | 店方 maintenance panel / 門鎖本機頁面上的事件順序 |
| 核准片段 | 審查會核准調出的有限原始片段 |

不要讓角色一直說 raw / sync / report layer / Narrative Anchor Rule。內部規則可以存在於 writer note，但玩家只需要聽懂這句：

> 摘要抓到的，不一定是第一個人。它抓的是最好對得上的那個。

---

## 3. 每 beat 共同 QA

每一 beat 寫完都要檢查：

- 這一段是否只推進一個主要問題？
- 是否有至少一個具體可見物，而不是只有台詞說明？
- 是否把主線伏筆留白，而不是角色提前解釋？
- 是否讓相馬、早坂、神谷、黑瀨各自用自己的方法看證據？
- 是否避免讓雨宮像攻略提示？
- 是否避免讓 KAGAMI 像任意造假系統？
- 是否讓玩家感到自己在拆 summary，而不是聽相馬朗讀答案？

# Beat 0：冷開場 — 摘要的乾淨故事

## Beat 功能

讓玩家先被 KAGAMI 摘要說服，再用 5～10 秒的人性鏡頭提醒：這份摘要會壓到活人。

這段必須短、乾淨、冷。它不是懸疑預告片，也不是哭戲開場。

## 預估時長

2～3 分鐘。

## 場景單位

### Scene 0A：KAGAMI UI 黑底摘要

地點：黑底 UI / KAGAMI 摘要畫面。  
角色：無真人，只顯示資料。  
目的：建立官方故事。

畫面順序建議：

1. 死者：增田圭。
2. 地點：吉祥寺咖啡館「雨鐘」。
3. 推定案發時間：23:08～23:12。
4. 門鎖事件：23:07:50，Staff Credential，三宅蒼太。
5. 監視器截圖：三宅靠近後場。
6. 打卡紀錄：三宅當晚值班。
7. 建議：主要嫌疑人，三宅蒼太。
8. UI 角落極短閃過：`merge delay: 89.7s`。

注意：`89.7s` 不要放大，不要配特殊音效。玩家第一次可以完全忽略。

Writer note：23:07:50 是摘要畫面標在 Staff Credential 上的時間，不是本機 raw 秒數。本機頁面只會在 Beat 8 顯示事件順序。

### Scene 0B：人性鉤子

地點：審查會 / 警署會面室外走廊。  
角色：三宅母親。  
目的：讓玩家知道 summary 壓到的是一個家庭。

畫面：

- 黑底 UI 顯示「主要嫌疑人：三宅蒼太」。
- 轉場前聽見塑膠袋被捏皺的聲音。
- 三宅母親坐在會面室外，手裡飯糰袋被捏得變形。
- 她沒有哭，只是一遍遍把袋口摺好，又摺壞。
- 下一秒切回相馬桌上的 KAGAMI 摘要副本。

## 玩家此時應理解

- 官方故事看起來很完整。
- 三宅看起來很危險。
- 這不是抽象系統問題，而是會影響一個普通家庭。

## 不要做

- 不要讓三宅母親說「我兒子不會殺人」。這句留到 Beat 2。
- 不要讓 UI 看起來邪惡或故障。
- 不要解釋 90 秒。
- 不要出現藍傘、金木犀、`ZW_A16.lock`。

## Exit Condition

玩家帶著「summary 很完整，但這會毀掉一個人」的感受進入相馬事務所。

# Beat 1：相馬事務所日常 — 年輕偵探的入口

## Beat 功能

建立相馬的初始狀態：他相信證據、相信整理好的資料能保護人，但仍不夠成熟。早坂作為法律與程序入口登場，同時用紙本習慣對照相馬對 summary 的信任。

## 預估時長

6～8 分鐘。

## 場景單位

### Scene 1A：相馬的小事務所

地點：相馬事務所，清晨雨聲。  
角色：相馬律。  
目的：基礎操作教學與角色建立。

可調查物：

| 物件 | 回應方向 | 功能 |
|---|---|---|
| 桌上舊委託單 | 相馬按時間排列 | 顯示他喜歡秩序與資料 |
| 壞掉的咖啡機 | 只吐出一點熱水；相馬仍測水溫 | 預告咖啡設備 / 出杯紀錄會在本章出現，但不要寫成餘溫定時器 |
| 便利店罐咖啡 | 相馬的生活拮据 | 人物日常 |
| KAGAMI 摘要副本 | 進入證物欄 | 正式證物入口 |

相馬可有一句偏信 summary 的台詞，但全章前半只保留這一次。

推薦對話：

> 相馬把摘要攤在桌上。  
> 「時間、門、鏡頭，三條線都對上了。」  
> 早坂沒抬頭，只把便利店收據壓在紙角。  
> 「那你先去店裡走一遍。走完還對得上，再信它。」

### Scene 1B：早坂進場

地點：同上。  
角色：相馬、早坂茜。  
目的：引入委託與程序。

早坂行為要比台詞重要：

- 她把便利店收據貼進紙本筆記。
- 她不反科技，但習慣保留紙本來源。
- 她不說「summary 不是現場」這種 thesis 句，只要求相馬去現場走一遍。

## 玩家此時應理解

- 相馬不是反 KAGAMI 的人。
- 他一開始相信摘要。
- 早坂更在意資料來源與保全鏈。
- 案件入口不是偵探私闖，而是法律程序。

## 證物更新

正式證物：

- KAGAMI 摘要副本。

人物筆記：

- 相馬：會用資料感抵抗不安。
- 早坂：紙本收據癖，重視來源。

## 不要做

- 不要讓早坂講出本章答案。
- 不要讓相馬一開始就懷疑 KAGAMI。
- 不要把壞咖啡機變成搞笑段過長。

## Exit Condition

早坂提出委託，帶相馬進入三宅母親與審查會程序。

# Beat 2：委託與程序入口 — 三宅母親求助

## Beat 功能

讓三宅母親正式委託，建立三宅作為普通人的生活壓力。同時明確審查會不是教學演習，而是「保住重新看證據的權利」。

## 預估時長

8～10 分鐘。

## 場景單位

### Scene 2A：母親委託

地點：相馬事務所 / 早坂律師事務所。  
角色：相馬、早坂、三宅母親。  
目的：接受委託與建立情感動機。

母親不要只哭訴。她要帶著生活細節：

- 飯糰袋。
- 胃藥或保溫杯。
- 提到三宅常把店裡準備丟掉的蛋糕邊帶回家。
- 說三宅常忘記自己吃飯。

她可以說三宅不會殺人，但相馬不能把它當證據。

可用短對話：

> 母親：「他會說謊。小孩子都會說謊。可是……殺人那種事，他做不到。」  
> 相馬看了一眼摘要。  
> 「這句話不能當證據。」  
> 早坂：「所以我們才需要去找證據。」

### Scene 2B：審查會入口與程序摩擦

地點：審查會入口 / 書記官窗口。  
角色：相馬、早坂、書記官。  
目的：建立相馬合法入場邊界。

必須演出程序摩擦：

1. 書記官先攔下相馬。
2. 早坂出示辯方聘任、法院核准、第三方 evidence analyst 文件。
3. 相馬被提醒：他不是警察，不能任意要求 raw。
4. 早坂簡短說明「限定調出」標準。

可用程序說明：

> 早坂：「如果我們能提出至少兩項獨立矛盾，而且都指向同一裝置的時序可信度，審查會可以核准限定調出。」  
> 相馬：「不是打開全部資料？」  
> 早坂：「只打開那個裝置、那個時間段、那個爭議欄位。」

### Scene 2C：法律壓力

地點：審查會外走廊。  
角色：相馬、早坂。  
目的：設定當日 stakes。

必須說清楚：

> 早坂：「今天如果不能證明摘要時間線有具體矛盾，三宅會被正式起訴。起訴後資料會封存，我們能要求的片段會少很多。」  
> 相馬：「也就是說，今天不是要贏辯論。」  
> 早坂：「是要保住重新看證據的權利。」

## 玩家此時應理解

- 三宅有生活壓力，但還沒有被洗白。
- 相馬能進入審查會有程序根據。
- raw data 不是隨便能看。
- 這場審查會有法律後果。

## 證物更新

正式證物：

- 三宅母親的委託書。
- 三宅班表影本。

人物筆記：

- 三宅母親胃不好。
- 三宅常帶蛋糕邊回家。

## 不要做

- 不要讓母親提供完整不在場證明。
- 不要把 限定調出 寫成長篇制度課。
- 不要讓相馬像已有調查特權。

## Exit Condition

相馬與早坂取得合法調查入口，準備前往「雨鐘」。

# Beat 3：第一次現場調查 — 雨鐘咖啡館

## Beat 功能

讓玩家第一次看見現場，收集表面證據，同時提前可視化後面會被追問的問題：三宅 routine、L 型後場、閉店 SOP、23:20 發現延遲、第二杯咖啡、北見 ambient mention。

## 預估時長

20～24 分鐘。

## 場景單位

### Scene 3A：進入雨鐘

地點：吉祥寺咖啡館「雨鐘」前場。  
角色：相馬、早坂、店長高瀨、片瀨。  
目的：建立店的生活感與雨夜氛圍。

玩家可見物：

- 傘架上的藍色透明傘。
- 金木犀拿鐵看板。
- 收銀台。
- 閉店清潔用品。
- 片瀨趕末班車的焦躁。

藍傘只做空鏡，不討論。金木犀讓相馬手停半拍即可。

### Scene 3B：閉店 routine 監視器

地點：店內監視器回放 / 收銀台旁。  
角色：相馬、早坂、店長。  
目的：讓玩家知道三宅進出後場是正常工作的一部分。

20～30 秒畫面：

1. 三宅補紙杯。
2. 去後場拿清潔用品。
3. 到倉庫門口取紙箱。
4. 回前場擦桌。
5. 去後巷倒垃圾。

注意：這不是洗白。這只是讓「三宅靠近後場」不等於犯罪行為。

### Scene 3C：L 型後場與視線遮蔽初見

地點：後場走廊、內側倉庫入口。  
角色：相馬、早坂、店長。  
目的：讓玩家早早建立空間感。

可調查點：

| 調查點 | 玩家看到 | 功能 |
|---|---|---|
| 後場入口 | 由前場通往走廊 | 三宅能正常進入 |
| 清潔用品架 | 位於走廊前段 | 三宅 23:06 的目的地 |
| 蛋糕盒 / 紙杯箱 | 位於走廊前段 | 解釋三宅小謊 |
| 半掩防火門 | 通往內側倉庫 | 視線遮蔽 |
| 高貨架 | 遮擋深處 | 解釋看不到屍體 |
| 內側感應燈 | 需深入才會亮 | 23:20 發現屍體的前置 |

此時不要讓相馬直接推理三宅看不到屍體。先讓玩家記住空間。

店長看見相馬注意到後場舊掛鐘時，可以補一句生活感：

> 「那個鐘常慢。前任老闆留下的，我一直沒丟。」

這句只讓舊掛鐘合理存在，不要讓店長說出案發時間。

### Scene 3D：閉店維護 SOP 白板與紀錄冊

地點：後場白板 / 前場收銀台。  
角色：相馬、店長。  
目的：提前可視化 23:20 發現延遲。

白板內容：

- ✅ 22:50 閉店流程開始。
- ✅ 22:52 後場門鎖維護模式 ON。
- ✅ 收銀關帳開始。
- △ 小額差額確認中。
- ✅ 清潔用品回收。
- ✅ 蛋糕盒與紙杯箱整理。
- ⬜ 維護紀錄冊確認。
- ⬜ 關閉維護模式。

前場收銀台旁有尚未蓋章的維護紀錄冊。

這不必進正式證物欄，可作人物 / 現場筆記。後面 23:20 店長發現屍體時回收。

### Scene 3E：收銀紀錄與第二杯咖啡

地點：收銀台。  
角色：相馬、店長 / 片瀨。  
目的：建立增田等待第二人的線索。

收銀紀錄顯示：

- 增田點了兩杯咖啡。
- 第二杯杯套 / 備註為 `K.`。

注意：`K.` 不是完整姓名。店內可有用首字母標杯的習慣；不能讓警方顯得蠢。

### Scene 3F：北見 ambient mention

地點：前場 / 後場入口。  
角色：片瀨或店長。  
目的：讓外包維護人員在空氣中早早存在。

可用對話：

> 片瀨：「門鎖那邊今天也有預約嗎？每次都挑快關店的時候。」  
> 店長：「說是確認紀錄，很快就走。」

不進證物欄。

### Scene 3G：聲音環境種子

地點：前場 / 後場。  
目的：為致命打擊聲音遮蔽埋種子。

要自然出現：

- 洗杯機 / espresso machine backflush 排氣聲。
- 收銀抽屜彈開與硬幣盤碰撞聲。
- 雨打後巷鐵門作背景。
- 片瀨趕末班車，店長對帳。

不要讓角色逐項解釋「這些聲音會遮蔽殺人聲」。

### Scene 3H：增田使用後場的表面理由

地點：後場小辦公角落 / 收銀台旁。  
角色：相馬、早坂、店長。  
目的：補齊「增田為什麼能在咖啡館後場」的生活邏輯。

店長說明：

- 增田過去曾因雨鐘的門鎖試點資料，來店裡做過幾次資料確認。
