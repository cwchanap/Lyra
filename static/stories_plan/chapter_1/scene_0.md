# Scene 0: 開場

[場景：相馬律偵探事務所，吉祥寺附近某棟大廈三樓，夜。細雨打在窗玻璃上，桌上擺著幾疊文件與空咖啡杯。牆上一台小型電視正播放新聞，畫面是東京都市俯瞰與滑順的宣傳音樂。]

**旁白**：東京，十一月。

**旁白**：今年第一場入冬的冷雨，從傍晚就沒有停過。

**播報員**：KAGAMI 智慧城市系統今日宣布，將進一步擴大東京都內合作據點。

**播報員**：門鎖紀錄、顧客動線分析、資料安全審查——

**播報員**：KAGAMI 承諾，讓每一份城市紀錄都精確、可追溯、不說謊。

[相馬律坐在辦公椅上，看著電視，表情沒有起伏。]

**相馬律**：不說謊。

[他拿起遙控器，關掉電視。]

**相馬律**：好大的口氣。

[辦公室恢復安靜，只剩雨聲和冷氣機的低鳴。]

[門被敲響，三聲，不快不慢。]

**早坂茜**：開著的？

**相馬律**：沒鎖。

[早坂茜推門進來，外套略微被雨沾濕，手裡夾著一個深色文件夾。]

**早坂茜**：還沒吃飯吧。

**相馬律**：你今晚不是來問我有沒有吃飯的。

**早坂茜**：我帶了委託人。在樓下等。

[相馬目光落在早坂手上的文件夾。]

**相馬律**：什麼案子？

**早坂茜**：殺人嫌疑。嫌疑人是一個二十歲的兼職店員。

**早坂茜**：KAGAMI 門鎖時間線指向他。警察幾乎確定了。

**相馬律**：那找我做什麼？

**早坂茜**：因為他的家人說他不可能做這件事。

**相馬律**：每個嫌疑人的家人都這樣說。

**早坂茜**：我知道。

[早坂把文件夾放到桌上。]

**早坂茜**：但這次我也覺得怪。

[相馬看了早坂一眼，然後看向文件夾。]

**相馬律**：讓他們上來吧。

[場景：相馬律偵探事務所，同一空間，片刻後。訪客椅上坐著一名年輕女性，二十四、五歲，穿著整齊但略顯疲憊。她雙手放在膝上，背挺得很直。]

**旁白**：若槻葵。若槻蓮的姐姐。

**若槻葵**：謝謝您願意見我。

**相馬律**：說說情況。

**若槻葵**：我弟弟叫若槻蓮。他在吉祥寺一間叫雨鐘的咖啡館打工。

**若槻葵**：昨天晚上，那間咖啡館的倉庫裡死了一個人。

**若槻葵**：警察說，門鎖紀錄顯示只有蓮在案發時間待在倉庫裡。

**相馬律**：然後？

**若槻葵**：蓮說，他進倉庫時那個人還活著。他們有幾句爭吵，然後他就出來了。

**若槻葵**：但警察不信。因為系統顯示他離開的時間，比被害者死亡時間晚了將近一分鐘。

[相馬律沒有說話，手指輕輕敲著桌面。]

**若槻葵**：蓮不是完美的人。他有小錯誤。

**若槻葵**：但他不會殺人。

**相馬律**：你說的小錯誤是什麼？

[若槻葵短暫停頓。]

**若槻葵**：他偷拿過店裡即將報廢的咖啡豆去賣。家裡欠著債，他想補貼一點。

**若槻葵**：這件事他不敢說，怕警察認為他品行有問題。所以他的證詞一直說不清楚。

**早坂茜**：但這和殺人是完全不同的事。

**若槻葵**：我知道。可是警察不這樣看。

[相馬律靠回椅背，短暫看向天花板。]

**相馬律**：被害者是誰？

**若槻葵**：增田圭。KAGAMI 公司的資料審查員。那天來咖啡館做例行資料安全檢查。

**相馬律**：蓮和他有什麼過節？

**若槻葵**：增田先生懷疑店內有人在賣顧客資料。他一直盯著蓮，因為蓮家裡欠債。

**若槻葵**：他們那天在倉庫確實吵過。蓮說了幾句氣話。

**若槻葵**：但那是吵架，不是殺人。

[早坂茜輕聲說。]

**早坂茜**：警察說紀錄已經證明一切了。可是蓮說他離開時，那個人還活著。

[若槻葵抬起頭看著相馬律，聲音平靜，但眼眶微紅。]

**若槻葵**：我不是在請您幫我說我弟弟是好人。

**若槻葵**：我只是請您去看一看。看看那份紀錄，是不是真的無懈可擊。

[相馬律沉默了幾秒。]

**相馬律**：早坂小姐，案件資料整理好了嗎？

**早坂茜**：初步資料在文件夾裡。

**相馬律**：好。

[他站起來，拿起掛在牆上的外套。]

**相馬律**：我們今晚去現場看。

[場景：東京，吉祥寺方向，深夜車內。細雨打在擋風玻璃上，雨刷規律地擺動。相馬律開車，早坂茜坐副駕駛座，文件夾攤在膝上，就著車內燈光看資料。車外是濕漉漉的東京夜景。]

**旁白**：十一月深夜的東京，這個時間路上人不多。

**旁白**：只有偶爾一兩台計程車，和路邊便利商店的冷白燈光。

[相馬律盯著前方，沉默開了一會兒車。]

**早坂茜**：你剛才關電視的時機很準。

**相馬律**：什麼？

**早坂茜**：KAGAMI 的廣告。「紀錄不說謊。」你一聽到這句就關了。

**相馬律**：那種話聽了讓人煩躁。

**早坂茜**：你煩躁起來倒是挺快的。

[相馬沒有回應，只是換了個車道。]

**早坂茜**：你讀過 KAGAMI 嗎？系統架構那一塊。

**相馬律**：沒有。我只知道它讓警察覺得自己可以少做一半的工作。

**早坂茜**：這樣說有點不公平。它確實減少了很多人工誤差。

**相馬律**：也新增了幾種人工誤差。

**早坂茜**：比如？

**相馬律**：比如「系統說的，所以一定對」。

[早坂茜沒有立刻反駁，低頭翻了翻文件。]

**早坂茜**：若槻蓮的紀錄，你怎麼看？

**相馬律**：智慧門鎖說他在倉庫裡。手錶死亡紀錄說被害者在那段時間死亡。

**相馬律**：如果這兩份紀錄都沒問題，案子就是結了。

**早坂茜**：但是？

**相馬律**：但是若槻說他離開時那個人還活著。他偷過報廢咖啡豆，說不清楚自己的行動。

**相馬律**：兩件事都是真的，但組合起來讓他看起來很像殺人犯。

**早坂茜**：你覺得哪裡有問題？

**相馬律**：還不知道。這才要去看。

[沉默片刻。雨聲稍微大了一點。]

**早坂茜**：你有沒有發現，你問若槻葵的第一個問題是「什麼案子」，不是「你弟弟有沒有做？」

**相馬律**：那不一樣嗎？

**早坂茜**：一般人第一個問的是「你弟弟是清白的嗎？」

**相馬律**：清不清白是我去查完才知道的事。問了也沒意義。

**早坂茜**：嗯。

[早坂茜把文件夾闔上，看向車窗外的夜景。]

**早坂茜**：你知道我為什麼每次帶委託人都來找你，不去找別的偵探嗎？

**相馬律**：因為我收費合理。

**早坂茜**：因為你不會先決定答案。

[相馬律沒有說話。]

**早坂茜**：大部分人拿到指向某個人的紀錄，就會開始找支持那份紀錄的證據。

**早坂茜**：你不這樣做。

**相馬律**：我只是懶得先做決定。

**早坂茜**：那也是一種方法。

[車子轉進一條小路，前方遠遠出現一排商店街的燈光。]

**旁白**：吉祥寺。東京西邊，安靜又帶點懷舊的街道。

**旁白**：平時到了這個時間，大多數店都關了。只剩幾間居酒屋還亮著燈。

**早坂茜**：雨鐘咖啡館在商店街裡面，再往前一個路口。

**相馬律**：嗯。

**早坂茜**：聽說它這個季節在賣金木犀拿鐵。限定款。

**相馬律**：喝咖啡是你的事。

**早坂茜**：我只是說，你可以先不要皺眉頭。到現場以後你的表情會自己管的。

[相馬律瞥了早坂一眼。]

**相馬律**：我沒有皺眉頭。

**早坂茜**：你從剛才就在皺了。

[場景：吉祥寺，雨鐘咖啡館外，深夜。咖啡館是一棟兩層樓的老建築，木頭招牌上的店名被雨淋得深沉。一樓窗戶透出昏黃的光。正門外拉著一道黃色警察封鎖線，幾盞移動燈具把入口照得很亮。路邊停著兩輛警察車輛。一名便衣刑警靠著電線桿站著，手裡夾著一支快燃盡的菸。]

**旁白**：雨鐘咖啡館。

**旁白**：今晚它不再賣咖啡。

[相馬律和早坂茜撐傘走近。那名刑警抬起頭，朝他們走了幾步。]

**黑瀨徹**：早坂律師。

**早坂茜**：黑瀨刑警。謝謝你讓我們進來。

**黑瀨徹**：別謝我。我只是覺得這案子太乾淨了。

[黑瀨把菸頭在鞋底磨掉，目光移向相馬律。]

**黑瀨徹**：你就是那個偵探？

**相馬律**：相馬律。

**黑瀨徹**：嗯。

[黑瀨打量了相馬兩秒，然後側身讓路。]

**黑瀨徹**：時間線是 KAGAMI 整理的。完整、清楚、沒有跳格。

**黑瀨徹**：若槻進倉庫，被害人死亡，若槻出來。三件事連在一起，沒有空隙。

**相馬律**：這種時候有沒有空隙，通常要自己去找。

[黑瀨看了相馬一眼，嘴角稍微動了一下。]

**黑瀨徹**：你知道嗎，太乾淨的時間線，通常是人擦過的。

**黑瀨徹**：進去看吧。別亂動證物。

[相馬律走向入口，在封鎖線下彎腰穿過去的瞬間——]

[——一陣風從後巷方向吹來，帶著一絲甜膩的氣味。]

[相馬律的腳步短暫停頓了一下。]

**早坂茜**：怎麼了？

**相馬律**：沒什麼。

[他繼續往裡走。]

[咖啡館的燈還亮著。空氣裡有金木犀拿鐵的香氣，混著咖啡渣和雨水的氣味。]

**旁白**：調查開始。
